# Navinkumar M - React Portfolio

Modern, responsive personal portfolio built with React, Vite, Tailwind CSS, and Framer Motion.

## Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- React Icons

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Project Structure

```txt
src/
  assets/
  components/
  sections/
  App.jsx
  main.jsx
```
